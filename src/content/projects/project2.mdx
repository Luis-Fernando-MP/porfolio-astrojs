### Requisitos para correr la aplicación

- Tener instalado [🔗 NodeJs](https://link-url-here.org) en su versión 18 o superior.
- Tener instalado [🔗 Mysql](https://link-url-here.org) en su versión 1.78 o superior.
- Tener instalado [🔗 Git](https://link-url-here.org) en su versión 2.32.0 o superior.

### Instalación y <b class='dg'>Ejecución</b>

1. Clone el repositorio en su máquina local:

```bash
git clone https://github.com/luJisJu/porfolio
```

2. Instalar las dependencias del proyecto:

```bash
pnpm install
```

3. Clone el repositorio en su máquina local:

```bash
pnpm run dev
```
